# ELSIP Prototype (Full System)

This archive contains a full prototype scaffold for the Ethiopia Labor & Skills Intelligence Platform (ELSIP).
It includes folders for:
- backend (Node.js + Express + PostgreSQL)
- ai (Rasa placeholder + OpenAI fallback)
- mobile (React Native Expo app scaffold)
- ussd (Africa's Talking / simple PHP endpoint)
- docker-compose.yml to run the stack locally (Postgres + API)

**Quick notes**
- This is a prototype scaffolding. Fill secrets in .env files before running.
- See individual READMEs in each folder for quick-start commands.
